# Bug Tracker Containerized Stack

A containerized multi-service application demonstrating how API, database, and reverse proxy communicate within a Docker network.

## Tech Stack
- FastAPI
- PostgreSQL
- Nginx
- Docker / docker-compose

## Goal
To understand:
- container networking
- service-to-service communication
- reverse proxy routing
- failure scenarios in distributed systems
